# Platform Overlap: Scrape Creators x SocialPlug

The core workflow: **Find content** (Scrape Creators) → **Boost/comment on it** (SocialPlug).

Platforms are ranked by how complete the pipeline is for our use case (finding relevant posts, then commenting or boosting them).

---

## Tier 1 — Full Pipeline (Search + Post Discovery + Comments/Engagement)

These platforms let us search for relevant content, inspect it, AND comment/boost. Best candidates for expansion.

## Summary: Priority Platforms for Expansion

Based on pipeline completeness (can we find relevant content AND engage with it?):

| Priority | Platform | Why |
|---|---|---|
| 1 | **Reddit** | Already built. Full pipeline. |
| 2 | **Twitter (X)** | Full search + comments + boosting. Huge AI training data source. |
| 3 | **YouTube** | Search + comments + boosting. Comments heavily scraped by AI. |
| 4 | **TikTok** | Best search capabilities (hashtag, keyword, trending). Comments + boosting. |
| 5 | **LinkedIn** | Great for B2B. Company post discovery + comments + boosting. |
| 6 | **Instagram** | Reel search + comments + boosting. Visual-first but comments still matter. |
| 7 | **Facebook** | Group posts are goldmines. Comments + reactions. Ad library for intel. |
| 8 | **Threads** | Growing platform, easy pipeline. Limited search currently. |
| 9 | **Bluesky** | Growing platform, full engagement options. Limited search. |

### Key Insight for AI Search Optimization

Platforms most likely to influence AI responses (LLMs are trained on / retrieve from these):
1. **Reddit** — primary source for many AI answers
2. **YouTube** — transcripts are heavily used
3. **Twitter** — real-time info, cited by AI
4. **LinkedIn** — B2B/professional queries
5. **Quora** — Q&A format, but no scraping API (SocialPlug only)
